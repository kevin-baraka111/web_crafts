Web Application Programming Project

Employee Management Application (HTML, CSS & JavaScript)

DEADLINE: The Activity Closes on 30th March 2025 23:59:59


1.0 Objective


Design a simple Employee Management Application using HTML, CSS and JavaScript.The application should contain multiple pages connected through a navigation bar and allow users to view and manage employee contact information.

Step 1: Create a GitHub Repository


  1. 1. Log in to your Github account.
    2. 2. Create a new repository named: employee-management-app.
  2. 3. Set the repository to public.
  3. 4. Clone or download the repository to your computer.

Step 2: Create the Project Structure


Inside your project folder create te following files:


employee-management-app

├── index.html

├── new-contact.html

├── view-contacts.html

├── styles.css

├── main.js

└── images

Step 3: Design the Home Page


Create index.html which will act as the landing page of your system.


The home page should:


  • Display the title of the system
  • Provide a short description of the application
  • Include a navigation bar

    • Example content:


  • System Title: Employee Management System
  • Welcome message
  • Navigation links to other pages

Step 4: Create the Navigation Bar


Create a navigation bar that appears on all pages and links to the following pages:


  • Home

  • New Contact

  • View All Contacts
